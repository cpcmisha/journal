# Contributing to Journal

Thank you for your interest in contributing.

## Development setup

Clone the repository:

```bash
git clone https://github.com/cpcmisha/journal.git journalnotes
```

Install dependencies:

```bash
composer install
npm install
```

Compile:

```bash
npm run build
```

## Coding style

- Follow PSR-12 for PHP.
- Use Vue 3 Composition/API conventions already present in the project.
- Keep Markdown files human-readable.
- Prefer small pull requests.

## Issues

Please include:

- Nextcloud version
- PHP version
- Browser
- Steps to reproduce

## Pull Requests

Pull Requests are welcome.

Please make sure:

- PHP syntax passes.
- JavaScript builds successfully.
- The application works on Nextcloud 34.

Thank you for helping improve Journal.
