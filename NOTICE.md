# Notices

Journal is developed and maintained by Miguel Torres (Misha).

## Historical origin

The project was initially developed from the open-source Diary application
created by Daniel Röhrig.

Journal has since evolved into an independent Nextcloud application with a new
application ID, PHP namespace, database schema, Markdown file storage,
YAML Front Matter, Nextcloud System Tags integration, global search,
internationalisation and a redesigned architecture.

Copyright notices and licence obligations from the original AGPL-licensed
source remain applicable to the portions derived from that project.
