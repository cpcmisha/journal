# Journal User Guide

## Overview

Journal is a daily Markdown journal for Nextcloud. Each entry is stored as a
regular Markdown file inside the user's `Journal` folder.

## Create or edit an entry

1. Open Journal from the Nextcloud app menu.
2. Select a date.
3. Write in the editor.
4. Journal saves the entry automatically.

Each date has one journal entry.

## Categories

An entry may belong to multiple categories.

Examples:

- Work
- Personal
- Ideas
- Study
- Health

Categories are stored in the YAML Front Matter of the Markdown file.

## Nextcloud tags

Journal uses native Nextcloud System Tags.

Tags added in Journal are also visible in Nextcloud Files and other compatible
applications.

## Search

Journal can search by:

- entry content
- date
- category
- Nextcloud tag
- wikilink
