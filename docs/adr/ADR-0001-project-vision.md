# ADR-0001: Visión del proyecto

**Estado:** Aceptado  
**Fecha:** 2026-07-12

## Contexto

El proyecto nació como una actualización de la aplicación Diary para versiones modernas de Nextcloud.

Durante la evaluación se identificó una necesidad más amplia: disponer de un espacio personal para escribir, recordar y conectar ideas dentro de Nextcloud.

## Decisión

Misha Journal evolucionará como un proyecto propio, manteniendo el reconocimiento y la licencia del proyecto original.

El fundador será también el primer usuario principal.

## Consecuencias

- Las funciones deberán responder primero a necesidades reales de uso.
- Se mantendrá una visión propia.
- El proyecto podrá continuar aunque tenga pocos usuarios externos.
- Se priorizará utilidad cotidiana sobre popularidad.
