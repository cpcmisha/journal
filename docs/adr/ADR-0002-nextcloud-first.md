# ADR-0002: Nextcloud primero

**Estado:** Aceptado  
**Fecha:** 2026-07-12

## Contexto

Existen numerosas aplicaciones de escritura y gestión del conocimiento, pero requieren otra plataforma o sincronización externa.

## Decisión

Misha Journal será una aplicación diseñada específicamente para Nextcloud.

Utilizará las capacidades de Nextcloud para:

- autenticación;
- usuarios;
- almacenamiento;
- permisos;
- temas;
- traducciones;
- notificaciones;
- archivos relacionados.

## Consecuencias

- No se buscará compatibilidad como aplicación web independiente.
- La experiencia deberá sentirse nativa de Nextcloud.
- La compatibilidad con nuevas versiones de Nextcloud será una prioridad.
