# Historias de usuario

## US-001 — Diario personal

Como usuario de Nextcloud, quiero escribir una entrada diaria para registrar lo que ocurrió, lo que aprendí y lo que deseo recordar.

## US-002 — Notas rápidas

Como profesional, quiero guardar pequeñas notas durante el día sin abrir otra plataforma.

## US-003 — Trabajo y vida personal

Como usuario, quiero registrar ideas laborales y recuerdos personales dentro de la misma aplicación, pero mantenerlos organizados.

## US-004 — Hábito diario

Como persona que quiere practicar journaling, quiero recibir recordatorios amables para escribir unos minutos cada día.

## US-005 — Editor visual

Como usuario que no conoce Markdown, quiero aplicar formatos mediante botones, insertar emojis y crear listas visualmente.

## US-006 — Usuario Markdown

Como usuario avanzado, quiero editar directamente el Markdown cuando lo prefiera.

## US-007 — Etiquetas

Como usuario con muchas entradas, quiero clasificarlas mediante etiquetas para encontrar temas relacionados.

## US-008 — Búsqueda

Como usuario que recuerda una palabra pero no la fecha, quiero localizar rápidamente cualquier entrada donde aparezca.

## US-009 — Páginas permanentes

Como usuario, quiero crear páginas que no dependan de una fecha, como proyectos, objetivos, personas o ideas.

## US-010 — Relaciones

Como usuario, quiero enlazar una entrada con otra página para conectar ideas relacionadas.

## US-011 — Backlinks

Como usuario, quiero saber qué entradas mencionan la página que estoy leyendo.

## US-012 — Recordatorios

Como usuario, quiero asociar un recordatorio a una nota para volver a ella en una fecha determinada.

## US-013 — Favoritos

Como usuario, quiero marcar páginas o entradas importantes para acceder rápidamente.

## US-014 — Recuerdos históricos

Como usuario habitual, quiero ver lo que escribí en esta misma fecha hace uno o varios años.

## US-015 — Libertad de salida

Como usuario, quiero exportar toda mi información en formatos abiertos y dejar de usar la aplicación sin perder datos.

## US-016 — Privacidad

Como usuario de Nextcloud, quiero que mis notas permanezcan en mi servidor y no se envíen a servicios externos.

## US-017 — Exportación

Como usuario, quiero exportar entradas individuales o todo mi diario en Markdown, HTML, PDF o JSON.

## US-018 — Imagen del día

Como usuario, quiero añadir opcionalmente una imagen a mi entrada diaria para conservar un recuerdo visual.

## US-019 — Estado de ánimo

Como usuario que practica journaling, quiero registrar opcionalmente cómo me sentí durante el día.

## US-020 — Uso durante años

Como usuario, quiero que mis entradas sigan siendo accesibles después de actualizaciones de Nextcloud y de la propia aplicación.
