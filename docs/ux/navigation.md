# Navegación principal

La navegación prevista de Misha Journal será:

1. Inicio
2. Hoy
3. Diario
4. Páginas
5. Etiquetas
6. Favoritos
7. Recordatorios
8. Buscar
9. Configuración

## Principios

- “Hoy” debe estar siempre accesible.
- La navegación no debe crecer indefinidamente.
- Las funciones secundarias pueden aparecer dentro de cada sección.
- Debe funcionar correctamente en escritorio y móvil.
