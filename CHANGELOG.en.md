# Journal

## 0.2.0

The first public release of Journal.

**What's new**

- Rich-text editing powered by Nextcloud Text.
- Daily entries stored as Markdown files.
- YAML Front Matter metadata.
- Multiple categories.
- Nextcloud System Tags.
- Note inspector sidebar.
- Relations between notes.
- Wikilinks for connecting pages.
- Relations explorer.
- Search across content, dates, categories, tags and wikilinks.
- Date navigation.
- Markdown export.
- Individual and combined PDF export.
- Emoji support.
- English, Spanish and German translations.

**Fixes**

- Isolated PHP dependencies to prevent conflicts with Nextcloud.
- Fixed the WebDAV check when the application is enabled.
- Fixed conflicts involving Dompdf, TCPDF, CommonMark and PSR Log.
- Fixed external links being incorrectly interpreted as wikilinks.
- Improved PDF generation and PDF merge stability.

Journal entries remain accessible as regular Markdown files inside Nextcloud storage.
