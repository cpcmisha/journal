# Journal Roadmap

Journal follows a stability-first development approach.

The first public release focuses on daily writing, portable Markdown storage,
organization and note relationships. Future development will be guided by
real-world usage and community feedback.

## 0.2.x — Stability and feedback

- Fix issues reported by users
- Improve compatibility with supported Nextcloud releases
- Refine desktop, tablet and mobile layouts
- Improve accessibility and keyboard navigation
- Improve translations and documentation
- Strengthen automated testing and release validation

## 0.3 — Planned improvements

The exact scope will be selected after reviewing user feedback.

Potential areas include:

- Improved backlinks and note relationships
- Calendar-oriented navigation
- Reminders for daily writing
- Search and filtering improvements
- Additional metadata options
- Improved dashboard integration

## Future ideas

These ideas are exploratory and are not commitments:

- Templates
- Statistics
- Graph visualization
- Additional export options
- Optional integrations with other Nextcloud applications

## Project principles

Journal will continue to prioritize:

- User ownership of data
- Native Markdown files
- Compatibility with Nextcloud
- A simple and focused interface
- Portability
- Privacy
- Stability over feature growth
